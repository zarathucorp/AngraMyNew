## 序言
AngraMainyu —— 挑战现有秩序的破坏性精神。  
我们将此名称转化为Zarathu的创造能量，  
在单纯破坏之外建立新的秩序与美。

我们都是**为全人类谋利益的艺术家**。  
发挥各自的个性，创造美，让世界更美好。

### Logo
<img src="img/Zarathu_Pattern.webp" width="300">
  
  我们的Logo受到**单图案非对称铺砖**的启发。  
同一基本图案重复，但绝不形成完美对称，完美表达了Zarathu的哲学。

- **单一图案**：我们都是人类，并致力于造福全人类。  
- **非对称性**：每个人都表达自己的个性。  
- **无限创造**：同样的人展现不同个性，填满世界，持续创造。

Zarathu汇聚拥有独特个性的艺术家，共同创造无尽美丽的世界。

## 经文
本仓库的经文是尼采的 **《查拉图斯特拉如是说》**。  
Zarathu社区的哲学基于此书的教导，  
通过仪式性里程碑和记录体系实现制度化与传承。

- “真理不在封闭的门内，而向所有人开放。”（前篇）
