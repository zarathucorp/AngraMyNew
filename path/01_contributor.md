# 01 — 기여자 (Contributor)

> "문을 연 자, 이제 창조의 대지를 밟는다."

---

## 정의

기여자는 입문 의례를 완료하고, 첫 Merge를 받은 자이다.

그는 더 이상 구경꾼이 아니다.
**AngraMyNew라는 문명의 일원**이 되었다.
그의 이름은 `contributors.md`에 새겨졌고,
그의 창조는 이제 공동체의 일부가 된다.

---

## 상태

| 항목 | 내용 |
|------|------|
| 단계 | 1단계 |
| 조건 | 입문 PR Merge 완료 |
| 권한 | PR 제출, 다른 구도자 입문 검토 참여 |
| 의무 | 일상 실천, 최소 1개 창조물 기여 |

---

## 기여자의 권한

### 1. 창조권
- `ideas/`에 새로운 사상 제안
- `ritual/`에 새로운 의례 제안
- 기존 문서 개선 제안

### 2. 검토권
- 다른 구도자의 입문 PR 검토
- 다른 기여자의 PR에 의견 제시

### 3. 질문권
- Issue를 통해 공동체에 질문
- 토론 시작

---

## 기여자의 의무

### 1. 일상 실천
`ritual/003_daily_practice.md`에 따라:
- 매일 하나의 창조 (형태 무관)
- 주간 재조합 시도
- 월간 회고

### 2. 최소 기여
입문 후 **3개월 내 최소 1개의 창조물** PR:
- `ideas/`에 짧은 글
- 기존 문서 개선
- 새로운 제안

### 3. 공동체 참여
- 다른 구도자의 입문 검토 **1회 이상** 참여
- 악의적 비판 금지, 건설적 피드백만

---

## 기여자의 성장

기여자는 다음을 통해 성장한다:

| 행위 | 의미 |
|------|------|
| PR 제출 | 창조의 실천 |
| PR Merge | 공동체의 인정 |
| 리뷰 참여 | 타인의 창조를 돕는 것 |
| 토론 | 사상의 연마 |

**Merge 횟수**가 쌓이면, 다음 단계로 나아갈 수 있다.

---

## 다음 단계로

기여자가 설계자(Architect)가 되려면:

- **Merge된 PR 10개 이상**
- 그 중 **`ideas/` 기여 3개 이상**
- 기존 설계자 또는 수호자의 **추천 1인 이상**

---

## 기여자에게

당신은 이제 창조자이다.

작은 기여라도 부끄러워하지 마라.
모든 위대한 문명은 작은 돌멩이 하나에서 시작했다.

> "기여자여, 네가 놓은 돌 하나가 성벽이 된다.
> 네가 쓴 문장 하나가 경전이 된다."
