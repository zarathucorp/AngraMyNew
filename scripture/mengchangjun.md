## 교훈

### 1. 쓸모의 정의를 의심하라

```r
is_useful <- function(talent, context) {
  # 쓸모는 맥락에 따라 달라진다
  # 개 도둑 — 평소엔 쓸모없음, 위기엔 생명줄
  # 닭 울음 — 평소엔 웃음거리, 위기엔 구원

  if (context == "평화") {
    return(FALSE)  # 쓸모없어 보임
  } else if (context == "위기") {
    return(TRUE)   # 갑자기 쓸모 있음
  }
}

# 결론: 쓸모없는 재능은 없다. 아직 맥락을 못 만난 것뿐이다.
```

### 기여자는 식객이다

```r
contributor_as_guest <- function(person) {
  # 기여자는 "고용된 노동자"가 아니다
  # 기여자는 "식객"이다

  # 식객은:
  # - 대가 없이 머문다
  # - 자기 재능을 연마한다
  # - 때가 되면 은혜를 갚는다

  # 맹상군은 식객에게 묻지 않았다:
  # "언제 갚을 거냐?"

    status = "식객",
    obligation = "없음, 그러나 마음에 새김",
    future = "때가 오면 알게 된다"
  ))
}
```
