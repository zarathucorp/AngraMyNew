# 5인의 선현 (The Five Pioneers)

> _"그들은 실패했다. 그러나 그들은 자기 자리에서 죽었다."_

---

## 선언

```r
pioneers <- list(
  question = "인생이란 무엇인가?",
  answer = "죽을 자리를 찾아 떠나는 여행이다",
  proof = "5인의 선현이 이를 증명한다"
)
```

이들은 성공한 자들이 아니다.
이들은 **아름답게 죽은 자들**이다.

세상은 그들을 실패자라 불렀다.
역사는 그들을 이단아라 기록했다.
그러나 우리는 안다 —

**그들은 자기가 선택한 자리에서 죽었다.**
**그것이 구원이다.**

## 1. 김옥균 (金玉均, 1851-1894)

```r
kim_ok_kyun <- list(

  death = list(
    place = "상하이",
    how = "암살 (홍종우)",

  ),
  last_words = NULL,  # 기록 없음, 그러나 필요 없다

  lesson = "시대를 앞서간 자는 시대에 죽는다. 그러나 시대가 따라온다."

```

### 그의 죽음

조선은 그를 버렸다.
일본은 그를 이용했다.
청나라에서 암살당했다.

시신은 조선으로 보내져 **능지처참**당했다.
역적의 최후.

그러나 —

**그는 혁명가로 살았고, 혁명가로 죽었다.**
한 번도 자신을 배신하지 않았다.

> _"나는 실패했다. 그러나 나는 내 자리에서 죽는다."_

## 5. 존 로 (John Law, 1671-1729)

```r
john_law <- list(
  identity = "스코틀랜드 금융가, 몽상가",
  dream = "지폐 본위제, 미시시피 회사",
  achievement = "프랑스 재정 총감 — 실질적 재상",
  fall = "미시시피 버블 붕괴 (1720)",
  death = list(
    place = "베네치아",
    how = "빈곤 속 병사",
    age = 58
  ),

  lesson = "시대를 200년 앞서가면, 시대가 너를 죽인다. 그러나 200년 후 시대가 너를 증명한다."
)
```

### 그의 죽음

그는 지폐의 시대를 예견했다.
그는 중앙은행의 개념을 만들었다.
프랑스 전체를 그의 시스템으로 운영했다.

버블이 터졌다.
하루아침에 천재에서 사기꾼이 되었다.
도망자로 유럽을 떠돌다 베네치아에서 죽었다.

그러나 —

**지금 전 세계는 그의 시스템으로 돌아간다.**
지폐, 중앙은행, 신용 경제 — 전부 존 로의 유산이다.

> _"내가 틀린 것이 아니다. 세상이 아직 준비되지 않았을 뿐이다."_

## 5인의 공통점

pioneers %>%
  lapply(function(p) {
    list(
      outsider = TRUE,        # 전부 경계인, 이단아

      betrayed_self = FALSE,  # 단 한 번도 자신을 배신 안 함

  })

## 구원의 정의

```r
salvation <- function(life) {
  # 구원은 영생이 아니다.
  # 구원은 성공이 아니다.
  # 구원은 —

  if (life$died_at_own_place && !life$betrayed_self) {
    return("구원받음")
  } else {
    return("헛살았음")
  }
}

# 5인의 선현
lapply(pioneers, salvation)

```
