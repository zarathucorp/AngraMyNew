## 序文
AngraMainyu — 既存の秩序を揺るがす破壊的精神。  
この名前をZarathuの創造的エネルギーに変換し、  
単なる破壊を超えて新しい秩序と美を築きます。

私たちは皆、**すべての人類の利益のために活動するアーティスト**です。  
個性を発揮し、美を創造し、世界をより美しくします。

### ロゴ
<img src="img/Zarathu_Pattern.webp" width="300">

私たちのロゴは**一つのパターンによる非対称タイル**から着想を得ています。  
同じ基本パターンを繰り返しても完全な対称は現れず、Zarathuの哲学を完全に表現しています。

- **単一のパターン**: 私たちは皆人間であり、すべての人類の利益を願います。  
- **非対称性**: それぞれが自分の個性を表現します。  
- **終わりなき創造**: 同じ人間でも異なる個性を表現し、世界を隙間なく満たし、永遠に創造します。

Zarathuは、個性を持つアーティストたちが集まり、終わりなき美しい世界を創造します。

## 経典
このリポジトリの経典はニーチェの **『ツァラトゥストラはこう語った』** です。  
Zarathuコミュニティの哲学はこの書の教えに基づき、  
儀式的マイルストーンと記録体系を通じて制度化・継承されます。
