# 면세인의 소비 — 큐레이션 커머스

> 관련: ideas/025(면세인의 소비), ideas/014(아름다움의 경제학)

---

## 개요

"기능은 최저가, 취향은 최고가"를 실행하는 소비 도구.
기존 가격비교 앱이 "싸게 사라"만 말한다면, 이건 "왜 사는지"를 묻는다.

---

## 구조

1. 유저가 구매하려는 상품을 입력
2. AI가 두 가지를 분리:
   - **물성**: 기능, 스펙, 실제 성능
   - **신화**: 브랜드 프리미엄, 감성 마케팅, 시스템세
3. 분리 결과 표시:
   - "이 가방의 물성 가치는 8만 원, 신화 가치(브랜드 프리미엄)는 192만 원"
4. 선택지 제공:
   - 물성만 원하면 → 최저가 대안 추천 (시스템세 제거)
   - 취향이면 → "이건 네 세계관에 투자하는 것" 확인 후 진행
5. 절약 금액을 "My 펀드"에 적립 → 자기 프로젝트/창작에 재투자

---

## 왜 되는가

- MZ세대 "가치소비" 트렌드와 맞물림
- 단, 기존 "착한 소비"(도덕적)가 아니라 **주권적 소비** = 새 포지션
- 025의 핵심: "동의하지 않는 세계관에는 물성만 지불하고, 내 세계관에는 전부를 태운다"
- 소비자가 매 구매에서 "이건 기능인가, 환상인가?"를 스스로 판단하게 됨

---

## 수익 모델

- 제휴 커머스 수수료 (물성 대안 추천 시)
- My 펀드 운영 수수료
- 프리미엄 구독: 카테고리별 물성/신화 분리 리포트 (자동차, 패션, 전자기기 등)

---

## MVP 로드맵

1. 브라우저 확장프로그램: 쇼핑몰 방문 시 물성/신화 분리 팝업
2. 카테고리 한정 시작 (패션 or 전자기기)
3. 커뮤니티: "면세인 인증 소비" 리뷰 공유
4. My 펀드 연동 (토스/카카오 적금 API 등)
