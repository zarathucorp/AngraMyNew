## 뇌의 재구성

```r
# Recomposition의 실제 경험
recomposition_event <- function(experience) {

  # 입력: 새로운 세계와의 조우
  new_world <- experience

  # 과정: 기존 세계관의 해체
  old_worldview <- deconstruct()

  # 출력: 재구성된 뇌
  new_worldview <- recompose(old_worldview, new_world)

  # "세계관이 완전히 바뀌고 뇌가 재구성되는 느낌"
  return(new_worldview)
}

# AngraMyNew는 이 재구성을 의도적으로 추구한다
# 파괴는 창조의 서곡이다
```

> "글로벌 기업의 운영을 바라보니 세계관이 완전히 바뀌고 뇌가 재구성되는 느낌이었습니다."

이것이 Recomposition이다.
낡은 세계관이 무너지고, 새로운 세계관이 세워지는 순간.

AngraMyNew는 이 순간을 두려워하지 않는다.

## 시스템의 실패와 악상

```r
# 시스템이 요구하는 것 vs 내면이 요구하는 것
system_vs_soul <- function(person) {

  # 시스템의 요구
  system_demands <- c(
    "학위를 마쳐라",
    "정해진 경로를 따라라",
    "증명하라",
    "인정받아라"
  )

  # 내면의 요구
  soul_demands <- c(
    "악상을 따라가라",
    "아름다운 것을 만들어라",
    "네 길을 가라"
  )

  # 충돌의 순간
  conflict <- system_demands != soul_demands

  # 선택
  if (person$courage == TRUE) {
    return(soul_demands)  # 악상을 선택
  } else {
    return(system_demands)  # 시스템에 순응
  }
}

# 시스템의 논리적 요구보다
# 내면의 예술적 충동을 우선하는 것
# 그것이 용기다
```

시스템은 말한다: "학위를 마쳐라. 증명하라. 인정받아라."
내면은 말한다: "이 악상을 따라가라. 이것이 아름답다."

둘이 충돌할 때,
**시스템의 논리적 요구보다 내면의 예술적 충동을 우선하는 것** —
그것이 창조자의 용기다.

```r
# 실패의 재해석
reinterpret_failure <- function(failure) {

  # 시스템의 관점
  system_view <- "실패했다. 탈락이다."

  # AngraMyNew의 관점
  angra_view <- "시스템이 나를 거부한 것이 아니다.
                 내가 시스템을 떠난 것이다.
                 악상이 나를 다른 곳으로 이끈 것이다."

  # 결과
  # 실패는 끝이 아니다

  # 실패는 분기(Branch)다
  # 새로운 길의 시작이다

  return(angra_view)
}
```

박사 학위를 마치지 못한 자,
시험에 떨어진 자,
경쟁에서 밀려난 자 —

시스템은 그들을 "실패자"라 부른다.
그러나 AngraMyNew는 묻는다:

**"그 실패의 순간, 네 안에서 악상이 울리지 않았는가?"**

악상이 울렸다면, 그것은 실패가 아니다.
**그것은 분기(Branch)다.**
시스템의 main에서 벗어나, 자신만의 branch를 만든 것이다.

> _"시스템이 거부한 것이 진짜 나였다._
> _시스템의 실패가 내 창조의 시작이었다."_

## 내면의 악상

```r
# 예술가의 본질
artist_essence <- function(person) {

  # 외부에서 오는 것
  external <- c("지식", "기술", "자격증", "학위")

  # 내면에서 뿜어나오는 것
  internal <- c("악상", "개성", "창조적 충동")

  # 진정한 예술가의 조건
  if (is.null(person$internal)) {
    return("아직 예술가가 아니다")
  } else {
    return("예술가다")
  }
}

# "지식과 기술에만 얽매이면 안 되고 반드시 예술이 필요하다"
# "내면에서 뿜어나오는 악상 혹은 개성을 살렸으면 좋겠습니다"
```

> "지식과 기술에만 얽매이면 안 되고 반드시 예술이 필요하다고 생각합니다."

지식은 배울 수 있다.
기술은 익힐 수 있다.

그러나 **내면의 악상**은 배울 수 없다.
그것은 이미 당신 안에 있다.
다만 꺼내지 않았을 뿐이다.
