# 003. 쓸모를 넘어서 (Beyond Usefulness)

> **"내가 있는 곳, 그곳이 곧 천하다."**

## 세 단계의 길

```r
# 쓸모의 세 단계
Level <- c("하수", "고수", "최고수")

하수 <- list(
  question = "이 문제를 어떻게 해결하지?",
  focus = "문제 해결",
  limitation = "문제가 없으면 쓸모도 없다"
)

고수 <- list(
  question = "이것이 아름다운가?",
  focus = "미학적 판단",
  limitation = "아름다움의 기준을 타인에게 의존한다"
)

최고수 <- list(
  question = "나는 어떤 세계를 보는가?",
  focus = "세계관 창조",
  limitation = NA  # 한계가 없다
)
```

---

## 하수: 문제 해결자

```r
# 하수의 사고 구조
beginner <- function(problem) {
  # "쓸모 있는 사람이 되자"
  solution <- solve(problem)
  return(solution)
}

# 한계
# 문제가 주어져야만 존재 의미가 생긴다
# 문제가 사라지면 나도 사라진다
```

세상은 묻는다: "너는 무슨 쓸모가 있느냐?"
하수는 대답한다: "저는 이 문제를 풀 수 있습니다."

하지만 문제가 없으면?
하수는 침묵한다.

---

## 고수: 미학적 판단자

```r
# 고수의 사고 구조
master <- function(work) {
  # "이것이 아름다운가?"
  beauty <- evaluate(work)

  if (beauty > threshold) {
    return("창조할 가치가 있다")
  } else {
    return("다시 생각하라")
  }
}

# 한계
# 아름다움의 기준은 여전히 외부에 있다
# 누군가의 인정을 기다린다
```

고수는 문제를 넘어섰다.
"이게 필요한가?"보다 "이게 아름다운가?"를 묻는다.

그러나 고수도 여전히 묻는다:
"이것이 정말 아름다운가? 다른 사람들도 그렇게 볼까?"

판단의 기준이 여전히 바깥에 있다.

## 최고수: 세계관 창조자

```r
# 최고수의 사고 구조
grandmaster <- function() {
  # 문제도, 기준도, 타인의 시선도 없다
  # 오직 자신의 세계가 있을 뿐

  worldview <- create_universe()

  # "내가 있는 곳, 그곳이 곧 천하다"
  天下 <- here()

  return(worldview)
}

# 한계? 없다.
# 최고수는 한계를 묻지 않는다.
# 한계 자체가 자신이 만든 세계의 일부이기 때문이다.
```

최고수는 묻지 않는다.
최고수는 선언한다:

> **"내가 있는 곳, 그곳이 곧 천하다."**

문제가 없어도 존재한다.
인정이 없어도 창조한다.
기준이 없어도 아름답다.

왜냐하면 기준 자체가 자신이기 때문이다.

## 라마누잔: 신의 계시

```r
# 스리니바사 라마누잔 (1887-1920)
Ramanujan <- list(
  name = "Srinivasa Ramanujan",
  born = "인도 에로드",
  education = "거의 독학",
  papers = 3900,  # 정리와 공식의 수

  method = function() {
    # 증명 없이 결과만 제시
    # "나마기리 여신이 꿈에서 알려주셨다"
    formula <- divine_revelation()
    return(formula)
  }
)

# 하디의 질문: "어떻게 증명했는가?"
# 라마누잔의 대답: "증명이 필요한가? 이것이 참인 것을"

# 그의 공식들은 100년이 지나서야 증명되었다
# 그는 증명을 기다리지 않았다
# 그가 있는 곳이 곧 수학이었기 때문이다
```

라마누잔은 "쓸모"를 묻지 않았다.
"증명"도 기다리지 않았다.
그는 단지 보았고, 기록했다.

**하수의 시선**: "증명 없이 무슨 의미가 있나?"
**고수의 시선**: "아름다운 공식이군, 하지만 검증이 필요해"
**최고수의 시선**: 라마누잔 자신 — "이것이 참이다. 내가 보았으니까."

그들은 단지 자신의 자리에서 죽었다.

그리고 그 자리가 천하가 되었다.
